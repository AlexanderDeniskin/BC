# Excel Buffer Extended
Example, how to create an excel file with extended formating in BC cloud.

Any Excel file is a zip file containing a set of Xml files inside. The idea of this modification is to create the necessary set of Xml documents and add them to a zip archive.

The following features are available:
#### Cell Alignment
- Horizontal Alignment
- Vertical Alignment
- Indent
- Reading Order
- Shrink To Fit
- Text Rotation
- Wrap Text
- Justify Last Line

#### Cell Font
- Font Name
- Font Style
- Font Size
- Font Effects

#### Cell Borders

#### Cell Fill
- Background Color
- Patern Style
- Patern Color
- Fill Effects

#### Cell Comments
- Notes
- Note headers

#### Merge Cells

#### Auto Filter

#### Column Properties
- Width
- Outline Level
- Visibility
- Expand/Collapse

#### Row Properties
- Height
- Outline Level
- Visibility
- Expand/Collapse

#### Freeze Panes

#### Page Setup
- Paper size
- Page scale
- Page margins
- Page header and footer
